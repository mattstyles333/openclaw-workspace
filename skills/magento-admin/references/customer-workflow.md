# Customer Lookup Workflow

## Step-by-Step

1. **Navigate to Customers**
   ```
   https://sadmx.spex4less.com/monadminx472/customer/index/
   ```

2. **Search for Customer**
   - Email address (most reliable)
   - First/last name
   - Phone number
   - Account ID

3. **Review Customer Profile**

### General Info
- Name
- Email
- Phone
- Account status (Active/Inactive)
- Customer group (General, Wholesale, etc.)

### Addresses
- Default billing address
- Default shipping address
- Address book entries

### Orders
- Total orders placed
- Total spent
- Order history list
- Individual order links

### Account Activity
- Last login
- Account created date
- Password reset history

## Common Customer Checks

| Check | What to Look For |
|-------|------------------|
| Multiple accounts | Same email/name variations |
| High-value customer | Large total spent |
| Frequent returns | Order history pattern |
| Support history | Notes/comments on account |
| Address issues | Invalid/undeliverable addresses |

## Privacy Note

- Only access when necessary
- Don't share customer data unnecessarily
- Follow GDPR guidelines
- Log out when done
